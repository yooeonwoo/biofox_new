# BIOFOX KOL

BIOFOX KOL 프로젝트는 인플루언서 및 Key Opinion Leader (KOL) 관리를 위한 시스템입니다.

## 프로젝트 구조

- `biofox-kol/`: 주요 애플리케이션 코드
- `docs/`: 프로젝트 문서
- `n8n-workflow-builder/`: 워크플로우 자동화 도구

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev
```

## 기술 스택

- Next.js (App Router)
- TypeScript
- ShadCN UI
- Drizzle ORM
