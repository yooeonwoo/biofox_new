## Commission Management System PRD

### 프로젝트 개요
BIOFOX KOL 파트너들의 커미션을 자동으로 계산하고 관리하는 시스템을 구축하여 정확하고 투명한 수수료 정산을 실현한다.

### 목표
- 커미션 자동 계산 시스템 구축
- 투명한 정산 프로세스 확립
- 커미션 명세서 자동 생성 및 배포
- 수동 조정 기능으로 유연성 확보

### 주요 기능

#### 1. 커미션 계산 엔진
- 자동 계산 시스템
  - 월별 배치 처리
  - 다단계 커미션 구조 지원
  - 실시간 계산 미리보기
  - 계산 로그 및 감사 추적
- 계산 규칙 관리
  - 비주얼 규칙 빌더
  - 조건별 커미션율 설정
  - 특별 인센티브 규칙
  - 규칙 버전 관리
- 시뮬레이션 기능
  - What-if 분석
  - 변경 영향도 분석
  - 예상 커미션 계산

#### 2. 수동 조정 관리
- 조정 인터페이스
  - 금액 직접 입력
  - 조정 카테고리 분류
  - 증빙 서류 첨부
- 승인 워크플로우
  - 금액별 승인 단계
  - 승인자 자동 배정
  - 반려 사유 관리
- 일괄 처리
  - 다중 선택 처리
  - 그룹별 일괄 조정
  - 스케줄 지급 설정

#### 3. 명세서 시스템
- PDF 명세서 생성
  - 브랜드 템플릿 적용
  - 상세 내역 포함
  - 다국어 지원
  - 전자 서명 옵션
- 자동 배포 시스템
  - 이메일 대량 발송
  - 발송 상태 추적
  - 실패 시 재발송
  - 수신 확인 추적
- 명세서 아카이브
  - 월별/연도별 보관
  - 빠른 검색 기능
  - 다운로드 이력

### 기술 요구사항
- React 18+ with TypeScript
- PDF 생성 라이브러리 (jsPDF/puppeteer)
- 이메일 발송 시스템 연동
- 배치 처리 스케줄러
- Xano API 연동

### 성능 요구사항
- 10,000명 커미션 계산 10분 이내
- PDF 생성 3초/건
- 이메일 발송 1000건/분

### 보안 요구사항
- 금액 정보 암호화
- 접근 권한 엄격 관리
- 모든 변경 이력 추적
- 정기 보안 감사

### 완료 기준
- 계산 정확도 99.99%
- 자동 발송 성공률 95% 이상
- 정산 처리 시간 80% 단축
- 파트너 만족도 90% 이상
