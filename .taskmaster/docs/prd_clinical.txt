## Clinical Management System PRD

### 프로젝트 개요
BIOFOX 제품 사용 전후 임상 사진과 데이터를 체계적으로 관리하고 분석하는 시스템을 구축한다.

### 목표
- 임상 케이스의 체계적 관리
- 개인정보 보호 및 동의서 관리
- 임상 데이터 통계 및 분석
- 고품질 이미지 관리 시스템

### 주요 기능

#### 1. Clinical Case 관리
- 케이스 등록 및 관리
  - 환자 정보 (익명화)
  - 시술 정보 및 제품 사용 내역
  - Before/After 사진 매칭
  - 케이스 노트 및 코멘트
- 동의서 관리
  - 디지털 동의서 수집
  - 동의 상태 실시간 표시
  - 만료일 자동 알림
  - 동의서 갱신 프로세스
- 케이스 검색 및 필터
  - 제품별/시술별 분류
  - 기간별 필터링
  - 결과 만족도별 정렬
  - 태그 기반 검색

#### 2. Clinical Photo Viewer
- 고급 이미지 뷰어
  - 고해상도 이미지 지원
  - 핀치 줌 / 마우스 휠 줌
  - 이미지 비교 모드
  - 측정 도구 제공
- 워터마크 시스템
  - 자동 워터마크 적용
  - 텍스트/로고 워터마크
  - 위치 및 투명도 설정
  - 다운로드 시 적용
- 접근 권한 관리
  - 역할별 해상도 제한
  - 다운로드 권한 체크
  - 일일 다운로드 제한
  - 접근 로그 추적

#### 3. 통계 및 분석
- 데이터 집계 시스템
  - 실시간 통계 계산
  - 제품별 효과 분석
  - 시술자별 성과 분석
  - 기간별 트렌드 분석
- 시각화 대시보드
  - 인터랙티브 차트
  - 드릴다운 기능
  - 커스텀 리포트 생성
  - 데이터 내보내기
- 비교 분석 도구
  - A/B 테스트 결과
  - 제품 간 비교
  - 시술법 비교
  - ROI 분석

### 기술 요구사항
- React 18+ with TypeScript
- 이미지 처리 라이브러리
- Chart.js or D3.js for 시각화
- 이미지 압축 및 최적화
- Xano API 연동

### 성능 요구사항
- 4K 이미지 로딩 2초 이내
- 통계 계산 실시간 업데이트
- 동시 100명 이미지 뷰어 사용

### 보안 요구사항
- 환자 정보 완전 익명화
- 이미지 접근 권한 관리
- HIPAA 준수 고려
- 정기적 보안 감사

### 완료 기준
- 모든 임상 데이터 디지털화
- 동의서 관리 100% 자동화
- 통계 정확도 99% 이상
- 이미지 품질 손실 5% 이내
